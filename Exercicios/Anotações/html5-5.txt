                        Formatos de imagens pra web
JPEG: Consegue compactar uma imagem muito grande transformando ela em menor não tem formato a fundo tranparente. (Pouca qualidade de imagem)
Ex: Imagens grandes

PNG: Não permite animação mais permite (imagem com fundo transparente)
Ex: Imagens flutuantes na tela. (Mais qualidade de imagem)

GIF: É uma imagem animada.

                HTML 5 
HTML5 - É focado em semântica = Signficado
Ex: <u>Rua Rio grande do sul 2200</u>

Tag: <u></u> é uma forma não uma semântica, ela significa sublinhado

Tag: <adress>Rua rio grande do sul 2200</adress> - É uma tag semantica pois dá sentido, é uma tag de endereço.

            Formatação de Textos.
Negrito: <b>Algo</b> = Bolt = Sem semantica - Só existe forma.

Negrito: <strong></strong> = Com semântica

Italico: <i></i> = Sem semântica.

italico: <em>em</em> = Com semântica

                Marca Textos
Tag: mark - Cria um marca textos

        Letras Pequenas e Grandes

            CODE

            PRE

            CITAÇÕES
Citação Simples.
Tag: <q></q> = Serve para citar algo > Dando sentido á uma citação.
Ex: Você é muito 'inteligente' = Sem o sentido
Ex: Você é muito <q>inteligente</q> = com sentido.
OBS: '' é a mesma coisa de <q></q>.
OBS: Usar quando for uma citação.

Citação completa.
<blockquote></blockquote> = Quer dizer que serve para falar alguma coisa especifica.

                Links Externos

Tag: <a href =""> Serve para pôr um link externo.
Obs: Somente assim quando a pessoa clica no link ele entra so que na mesma página.

Tag: <a href ="" target ='_blank' rel = 'external'>
Obs: target = "_blank" é uma janela em branco

Obs: rel = 'external' quer dizer que ele é para ir pra fora da minha pagina.
